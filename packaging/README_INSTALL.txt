Tata Inventory App - Windows Setup

How to install:
1. Extract this ZIP anywhere on the target laptop/PC.
2. Double-click setup.bat.
3. After setup finishes, use the "Tata Inventory App" desktop shortcut.

What it installs:
- The dashboard app.
- A bundled Python runtime.
- Required Python libraries such as openpyxl.
- Current cached stock/material data needed by the app.

Default URL after launch:
http://127.0.0.1:8000
